# Minecraft Rust 启动器

这是一个独立的启动器程序，用于管理和启动 Minecraft Rust 游戏。

## 功能特性

- **现代化UI界面**：采用与游戏一致的现代风格设计
- **世界管理**：显示可用的世界列表，支持选择和启动
- **独立运行**：启动器与游戏分离，避免游戏卡顿影响菜单操作
- **直接启动**：点击世界即可直接启动游戏，跳过游戏内菜单

## 使用方法

### 编译启动器
```bash
cd launcher
cargo build --release
```

### 运行启动器
```bash
cargo run
```

### 启动器界面

1. **主菜单**
   - 单人游戏：进入世界选择界面
   - 设置：启动器设置（待实现）
   - 退出：关闭启动器

2. **世界选择**
   - 显示所有可用世界
   - 点击世界名称直接启动游戏
   - 创建新世界（待实现）
   - 返回主菜单

## 技术实现

### 启动器架构
- 使用 Bevy 引擎构建UI界面
- 独立的状态管理系统
- 通过命令行参数与主游戏通信

### 游戏启动流程
1. 启动器读取世界列表
2. 用户选择世界
3. 启动器调用主游戏程序，传递世界参数
4. 主游戏接收参数，跳过菜单直接进入游戏

### 命令行参数
主游戏支持以下命令行参数：
- `--world <世界名称>`：直接加载指定世界，跳过主菜单

## 文件结构
